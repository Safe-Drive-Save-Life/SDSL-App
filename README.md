# SDSL-App

## 🗒️ Summary
- 운전 중 차량 이상을 감지하여 적절한 조치 사항을 알려주는 서비스
- 산학협력체: (주) 인포카 

## ❔ Research Goals
- (기존 앱) 현재 차량의 고장 진단 결과만 제공 -> 미래 자신 차량이 어떤 상태일지 예측
- 예측 정도에 따라 운전자에게 적절한 조치 방법 제공
- 차량 전문 용어 최소화 -> 사용자가 이해하기 쉬운 UI 제공 

## 🛠 Tech
- Flutter, Dart
- TensorFlow Lite
- Flask, SQLite
- IDE Android Studio, Colab
